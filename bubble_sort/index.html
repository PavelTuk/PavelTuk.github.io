Отсортированный массив
  • Введите в поля "Минимальное значение" и "Максимальное значение" диапазон для генерации чисел. В поле "Количество элементов" соответственно задайте количество генерируемых элементов.
  • По умолчанию диапазон установлен от 1 до 999, а количество элементов равно 10.
  • Нажатие кнопки "Сгенерировать числа" запустит алгоритм, генерирующий массив чисел в соответствии с заданными параметрами (диапазоном и количеством элементов. Сгенерированный массив будет отображён в блоке "Исходный массив".
  • Нажатие кнопки "Отсортировать числа" запустит алгоритм сортировки "пузырьком".
  • Процесс и результат сортировки отображается в блоке "Отсортированный массив".
